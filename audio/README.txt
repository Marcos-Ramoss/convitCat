INSTRUÇÕES PARA O ARQUIVO DE MÚSICA

1. Verifique se o arquivo de música "Fim_Ou_Intervalo___De_Trás_Pra_Frente___Faz_Tempo__Ao_Vivo_Em_Brasília_(128k).m4a" está nesta pasta.

2. O nome do arquivo deve ser exatamente como está escrito acima, incluindo os underscores e parênteses.

3. O arquivo deve estar no formato M4A.

4. A música começará a tocar automaticamente quando o convite for aceito e continuará na página de escolhas.

5. Na página de escolhas, há um controle de volume no canto inferior direito que permite ajustar ou desligar a música.

Se você quiser usar uma música diferente:
- Coloque o novo arquivo nesta pasta
- Edite os arquivos script.js e choices.js para apontar para o novo arquivo
- Atualize o nome e formato do arquivo nas referências
